# SRB Healthcare Website

A professional healthcare infrastructure website for SRB Healthcare, featuring complete medical infrastructure solutions with a modern, responsive design.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional interface with red-blue-white theme
- **Interactive Elements**: Smooth animations, hover effects, and transitions
- **Gallery System**: Category-based image filtering with lightbox functionality
- **Contact Form**: Validated contact form with success feedback
- **FAQ Section**: Interactive accordion for frequently asked questions
- **Google Maps Integration**: Embedded map with location details

## Setup Instructions

1. **Extract Files**: Place all files in your web server directory
2. **Image Assets**: Add your actual images to the respective folders:
   - `images/logo.png` - Company logo
   - `images/hero-oxygen-plant.jpg` - Hero banner image
   - `images/about-hero.jpg` - About page hero
   - `images/services/` - Service-related images
   - `images/gallery/` - Gallery project images
   - `images/icons/` - Icon files
   - `images/team/` - Team member photos
3. **Update Contact Info**: Replace placeholder contact details in `contact.html`
4. **Configure Map**: Update Google Maps iframe with actual location coordinates
5. **Test Functionality**: Ensure all interactive elements work properly

## Customization

### Colors
Update CSS variables in `css/style.css`:
```css
:root {
    --primary-red: #c41e3a;
    --primary-blue: #0056b3;
    --primary-white: #ffffff;
}
```

### Fonts
Modify Google Fonts import in HTML files:
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```
